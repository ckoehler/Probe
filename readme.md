# Probe

# Keyboard Shortcuts

| Key       | Action                          |
| ----      | -----                           |
| q         | Quit                            |
| h, Left   | Previous Tab                    |
| l, Right  | Next Tab                        |
| j         | Next Probe                      |
| k         | Previous Probe                  |
| \<Enter\> | Show Details for selected probe |
