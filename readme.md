Probe is a ZMQ PUB/SUB monitor built with TUI.


![screenshot](assets/screen1.png)

# Keyboard Shortcuts

| Key       | Action                          |
| ----      | -----                           |
| q         | Quit                            |
| h, Left   | Previous Tab                    |
| l, Right  | Next Tab                        |
| j         | Next Probe                      |
| k         | Previous Probe                  |
| \<Enter\> | Show Details for selected probe |
